# 食事制限レシピ

食事制限のある人たちが美味しいものを作って食べるためのレシピサイト

## サービスURL

## 開発背景
食事制限のある方たちがいろいろな食べ方、レシピを共有できるサイトを作りたかった

## 主機能
・レシピの閲覧

・ログインすることでレシピの投稿、編集、削除ができる

・ログインすることでgoodボタンで評価できる

・チェックボックスにチェックを入れることで、各栄養量の多さから探すことができる

・レシピ検索で具体的に探すこともできる
